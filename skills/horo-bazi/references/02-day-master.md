# การวิเคราะห์ดิถี (Day Master) — คะแนน ความแข็ง/อ่อน และธาตุสำคัญ

ที่มา: "เคล็ดลับไม่มีในตำรา" วิชาดวงจีน 1 หน้า 8–13, 29–35, 38–39

**ดิถี = ราศีบนของตำแหน่งวัน** ใช้เป็นตัวตั้งในการวิเคราะห์ทั้งหมด

## ความสัมพันธ์ของธาตุอื่นกับดิถี (ความหมาย)

| ความสัมพันธ์ | ความหมาย |
|---|---|
| คู่ธาตุ (ธาตุเดียวกัน) | คนระดับเดียวกัน คนรอบตัว เพื่อน พี่น้อง หุ้นส่วน คู่แข่ง |
| ธาตุก่อเกิดเรา | การส่งเสริมเรา ทรัพยากร การศึกษา สถานภาพ สภาพจิตใจ การเลี้ยงดู ผู้สนับสนุน (หมายถึงแม่ด้วย) |
| ธาตุถ่ายเท (เราก่อเกิดมัน) | การลงมือทำ การพูด การแสดงออก วางแผน ความคิด ลงทุน ลงแรง ใช้จ่าย เดินทาง สิ่งที่อยากทำ (ดวงหญิงหมายถึงลูก) |
| ธาตุพิฆาตเรา (ธาตุอำนาจ) | สิ่งที่มีอำนาจเหนือเรา ภาระ หน้าที่ ตำแหน่ง หนี้สิน เจ้านาย แรงกดดัน ปัญหา (ดวงหญิงหมายถึงสามี ดวงชายหมายถึงลูก) |
| ธาตุโชคลาภ (เราไปพิฆาต) | โชคลาภ เงินทอง สิ่งที่ครอบครองได้ สิ่งที่เราไปควบคุม/เอาจากคนอื่น หมายถึงพ่อได้ (ดวงชายหมายถึงแฟนหรือภรรยา) |

## การให้คะแนนดิถี (แบบง่าย)

1. ใช้ธาตุของดิถีเป็นตัวตั้ง
2. นับเฉพาะช่องที่เป็น **คู่ธาตุ** และ **ธาตุก่อเกิดดิถี** ตามน้ำหนักช่อง (คะแนนเต็ม 9):

| | ยาม | วัน | เดือน | ปี |
|---|---|---|---|---|
| ราศีบน | 1 | 1 (ตัวดิถีเองนับด้วย) | 1 | 1 |
| ราศีล่าง | 1 | 1 | **2** | 1 |

3. ค่านี้ไม่มีวันเปลี่ยนตลอดชีวิต
4. แบ่งลักษณะดิถีตามคะแนน:

| คะแนน | ลักษณะดิถี |
|---|---|
| 1 ถึงไม่ถึง 2 | ดิถีตามกระแส หรือ สละดิถี (ดวงพิเศษ) |
| 2, 3, 4 | ดิถีอ่อน |
| 4.5, 5, 6, 7 ถึงไม่ถึง 8 | ดิถีแข็ง |
| 8, 9 | ดิถีเป็นกระแส (ดวงพิเศษ) |

ตัวอย่าง: ดิถีธาตุไฟ (丁) — ยาม 庚/子, วัน 丁/卯, เดือน 丁/酉, ปี 辛/丑 →
นับไฟกับไม้ได้ วันบน 1 + วันล่าง(卯 ไม้) 1 + เดือนบน(丁) 1 = **3 คะแนน → ดิถีอ่อน**

หมายเหตุ: นี่คือการวิเคราะห์เบื้องต้น เชิงลึกต้องดูการชง การมีราก คลังธาตุ ฯลฯ
(สอนในดวงจีน 2 — ไม่อยู่ในเอกสารนี้)

## ธาตุสำคัญ (ธาตุให้คุณ) — ตารางอย่างง่าย

ธาตุสำคัญ = ธาตุที่เสริมแล้วดวงสมดุล ชีวิตดีขึ้นทุกด้าน ถ้าถูกทำลายหรือขาดมาก
พลังงานตกต่ำ ดวงล้มได้

| ลักษณะดิถี | การปรับสมดุล | ธาตุสำคัญ |
|---|---|---|
| ดิถีอ่อน | ต้องการธาตุช่วยเหลือ | ธาตุก่อเกิดดิถี, คู่ธาตุกับดิถี |
| ดิถีแข็ง | ต้องการธาตุมาปรับสมดุล | ธาตุถ่ายเท, ธาตุโชคลาภ(ไปพิฆาต), ธาตุอำนาจ(มาพิฆาตดิถี) |
| ดิถีเป็นกระแส (ดวงพิเศษ) | เสริมธาตุรอบตัวของดิถี (เหมือนเป็นผู้นำ) | ธาตุก่อเกิด, คู่ธาตุกับดิถี, ธาตุถ่ายเท |
| ดิถีตามกระแส / สละดิถี (ดวงพิเศษ) | ไม่ชอบธาตุของตัวเอง | ธาตุถ่ายเท, ธาตุโชคลาภ(ไปพิฆาต), ธาตุอำนาจ(มาพิฆาตดิถี) |

## ตารางวิเคราะห์ดิถีอย่างละเอียด (โหมดคำนวณละเอียดของแอป)

กฎ HL จากหน้า 29–31:

- **HL6:** ถ้าดิถีอ่อน มีธาตุก่อเกิดโผล่บน (ปรากฏที่ราศีบน) → ธาตุอำนาจจะให้คุณด้วย
- **HL7:** ถ้าดิถีอ่อน มีธาตุก่อเกิด 2 ตัวขึ้นไป และมีธาตุโชคลาภ → คู่ธาตุจะให้โทษ
- **HL8:** ถ้าดิถีแข็ง มีธาตุก่อเกิดโผล่บน → ธาตุอำนาจจะให้โทษ
- **HL10:** ดวง 1–2 คะแนน (ดวงพิเศษ) จะไม่ชอบธาตุที่ทำลายธาตุสำคัญ

ตารางสรุป (หน้า 38–39):

| ลักษณะดิถี | เงื่อนไข | ธาตุสำคัญ |
|---|---|---|
| ดิถีอ่อน | มีธาตุก่อเกิดโผล่บน (ธาตุอำนาจให้คุณด้วย) | ก่อเกิดดิถี, คู่ธาตุกับดิถี, ธาตุอำนาจ |
| ดิถีอ่อน | มีธาตุก่อเกิด ≥2 ตัว และมีธาตุโชคลาภ (คู่ธาตุจะให้โทษ) | ก่อเกิดดิถี, ธาตุพิฆาต |
| ดิถีอ่อน | คะแนนไม่ถึง 3 และมีธาตุพิฆาตดิถีมากกว่า 1 หรือลงราก (= ดวงมีโรค) | ก่อเกิดดิถี, คู่ธาตุกับดิถี, ธาตุถ่ายเท |
| ดิถีอ่อน | อื่นๆ | ก่อเกิดดิถี, คู่ธาตุกับดิถี |
| ดิถีแข็ง | 4.5 คะแนน และมีธาตุพิฆาตดิถี ≥2 ตำแหน่ง (= ดิถีสมดุลแบบอ่อน) | ถ่ายเท, โชคลาภ(ไปพิฆาต) |
| ดิถีแข็ง | มีธาตุก่อเกิดโผล่บน (ธาตุอำนาจให้โทษด้วย) | ถ่ายเท, โชคลาภ(ไปพิฆาต) |
| ดิถีแข็ง | อื่นๆ | ถ่ายเท, โชคลาภ(ไปพิฆาต), อำนาจ(มาพิฆาตดิถี) |
| ดิถีเป็นกระแส | เสริมธาตุรอบตัวของดิถี — เรียงลำดับ: คู่ธาตุ, ธาตุถ่ายเท, ธาตุก่อเกิด(ปานกลาง) | ก่อเกิด, คู่ธาตุ, ถ่ายเท |
| ตามกระแส/สละดิถี | ตามโชคลาภ | ถ่ายเท, โชคลาภ(ไปพิฆาต), อำนาจ(มาพิฆาตดิถี) |
| ตามกระแส/สละดิถี | สละดิถีตามอำนาจ | โชคลาภ(ไปพิฆาต), อำนาจ(มาพิฆาตดิถี) |
| ตามกระแส/สละดิถี | สละดิถีตามธาตุถ่ายเท — เรียงลำดับ: คู่ธาตุ, ธาตุถ่ายเท, ธาตุโชคลาภ(ปานกลาง) | ถ่ายเท, โชคลาภ(ไปพิฆาต), คู่ธาตุอำนาจ |

## ดวงมีโรค

ดิถีอ่อนมาก (ไม่เกิน 3 คะแนน) ที่มีธาตุพิฆาตดิถีมากกว่า 1 หรือมีราก = **ดวงมีโรค**
→ ธาตุถ่ายเทจะเป็นธาตุให้คุณ (ธาตุรักษาโรค)

ตัวอย่างในตำรา: 壬/丁/丁/壬 บน, 子/亥/未/申 ล่าง (3 คะแนน พิฆาต 4 ตำแหน่ง ลงราก)
→ ธาตุให้คุณ ไม้ ไฟ และ ดิน

## ดวงร้อนจัดกับเย็นจัด (โหมดคำนวณละเอียด)

- **กรณี 1** เดือนล่างธาตุน้ำ (กุน ชวด ฉลู) → ธาตุให้คุณเพิ่ม **ไฟ**
  - ถ้าไฟให้คุณอยู่แล้ว หรือมีไฟในดวง ≥2 ตำแหน่ง → ไม่ต้องทำอะไร
  - ถ้าไฟให้โทษ → เปลี่ยนเป็นให้คุณ แต่ระบุเป็น ไฟ(ธาตุแก้)
- **กรณี 2** เดือนล่างธาตุไฟ (มะเส็ง มะเมีย มะแม) → ธาตุให้คุณเพิ่ม **น้ำ** (เงื่อนไขเดียวกัน กลับธาตุ)
- **กรณี 3** ดิถีดินแข็ง เกิดเดือนล่างน้ำ (กุน ชวด) → เพิ่ม ไฟ(ธาตุแก้)
- **กรณี 4** ดวงสละดิถี/เป็นกระแส เกิดเดือนล่างน้ำ (กุน ชวด ฉลู):
  - คะแนนน้ำ ≥ 4.5 → แก้ไม่ไหว, no action
  - คะแนนน้ำ < 4.5 → ถ้าไฟให้คุณหรือมีไฟ ≥2 ตำแหน่ง no action; ถ้าไฟให้โทษ เปลี่ยนเป็นให้คุณ ไฟ(ธาตุแก้)
- **กรณี 5** ดวงสละดิถี/เป็นกระแส เกิดเดือนล่างไฟ (งู ม้า แพะ): เหมือนกรณี 4 แต่กลับธาตุ (ไฟ↔น้ำ)

## ราศีแฝงของดิน (เฉพาะตำแหน่งเดือน, โหมดคำนวณละเอียด)

ตำแหน่งเดือน(ฤดู) ถ้าราศีล่างเป็นธาตุดิน จะให้พลังแฝงเพิ่ม
(ธาตุดินยังให้พลังสูงสุด พลังแฝงส่งเพียงครึ่งเดียว):

| ราศีล่าง | นักษัตร | ดินคลัง (ซาฮะ) | ดินปลายฤดู (ซาหวย) |
|---|---|---|---|
| 丑 ดิน- | ฉลู (ทิ่ว) | ทอง | น้ำ |
| 辰 ดิน+ | มะโรง (ซิ้ง) | น้ำ | ไม้ |
| 未 ดิน- | มะแม (บี่) | ไม้ | ไฟ |
| 戌 ดิน+ | จอ (สุก) | ไฟ | ทอง |
