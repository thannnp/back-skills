# ปฏิสัมพันธ์ราศี — ฮะ คัก ชง เฮ้ง ไห่ ผั่ว ซาฮะ ซาหุย หลักฮะ และธาตุแฝง

ที่มา: "เคล็ดลับไม่มีในตำรา" วิชาดวงจีน 1 หน้า 16, 41–53

กติกาการแสดงผลร่วม (จากสเปกในตำรา): ความสัมพันธ์เหล่านี้ **โชว์ทุกที่ยกเว้นพื้นดวง**
(เช็คระหว่าง วัยจร/ปีจร/เดือนจร/ดวงบ้าน กับดวงคน) และ **ต้องสัมพันธ์กับหลัก
ปี-เดือน-วัน-ยามเกิดเท่านั้น**

## ฮะราศีบน (Stem Combination) — แปรธาตุ

อั้มฮะ ใช้ธาตุแฝงตัวที่ 2, 3 เทียบ

| คู่ | แปรธาตุ |
|---|---|
| 甲 ไม้+ + 己 ดิน- | ดิน |
| 乙 ไม้- + 庚 ทอง+ | ทอง |
| 丙 ไฟ+ + 辛 ทอง- | น้ำ |
| 丁 ไฟ- + 壬 น้ำ+ | ไม้ |
| 戊 ดิน+ + 癸 น้ำ- | ไฟ |

## คัก — การพิฆาตราศีบน (Stem Clash)

เช็คด้านขวา ถ้าเจอด้านซ้าย **ตัวซ้ายคือคัก** ถ้าครบ 2 → ติดป้าย [คัก]

| ตัวคัก | ถูกคัก | | ตัวคัก | ถูกคัก |
|---|---|---|---|---|
| 甲 ไม้+ | 戊 ดิน+ | | 己 ดิน- | 癸 น้ำ- |
| 乙 ไม้- | 己 ดิน- | | 庚 ทอง+ | 甲 ไม้+ |
| 丙 ไฟ+ | 庚 ทอง+ | | 辛 ทอง- | 乙 ไม้- |
| 丁 ไฟ- | 辛 ทอง- | | 壬 น้ำ+ | 丙 ไฟ+ |
| 戊 ดิน+ | 壬 น้ำ+ | | 癸 น้ำ- | 丁 ไฟ- |

(คือธาตุพิฆาตขั้วเดียวกัน: ไม้คักดิน, ดินคักน้ำ, น้ำคักไฟ, ไฟคักทอง, ทองคักไม้)

## ชง — ราศีล่างปะทะ (Branch Clash)

ถ้าครบ 2 → ติดป้าย [ชง]

| คู่ชง | เหตุผล |
|---|---|
| 子 ชวด ↔ 午 มะเมีย | น้ำ- ปะทะ ไฟ- (กลุ่มแม่ธาตุ) |
| 丑 ฉลู ↔ 未 มะแม | คลังทอง ปะทะ คลังไม้ (กลุ่มคลังธาตุ) |
| 寅 ขาล ↔ 申 วอก | ไม้+ ปะทะ ทอง+ (กลุ่มพาหะ) |
| 卯 เถาะ ↔ 酉 ระกา | ไม้- ปะทะ ทอง- (กลุ่มแม่ธาตุ) |
| 辰 มะโรง ↔ 戌 จอ | คลังน้ำ ปะทะ คลังไฟ (กลุ่มคลังธาตุ) |
| 巳 มะเส็ง ↔ 亥 กุน | น้ำ+ ปะทะ ไฟ+ (กลุ่มพาหะ) |

ความหมายการชงรายหลัก: ตำราให้ไว้เฉพาะ **หลักปีจร × ปีเกิด** = ภาพรวมชีวิตจะ
เปลี่ยนแปลง การเดินทางไกล การต่างประเทศมีปัญหา (ที่เหลือ "ดูใน V6" — ไม่อยู่ในเอกสารนี้)

## เฮ้ง — การเบียดเบียน/ให้โทษ (เฉพาะราศีล่าง, รองจากชง) — 7 กลุ่ม

| กลุ่ม | สาเหตุ | ผล |
|---|---|---|
| 寅 ขาล, 申 วอก, 巳 มะเส็ง | กลุ่มพาหะธาตุพิฆาตกัน | มี 2 ใน 3: อารมณ์รุนแรง เข้ากับคนยาก; ครบ 3 = **ซำเฮ้ง** ร้ายกว่าชงมาก มักมีคนเกลียด เดินทางบ่อย โยกย้ายตลอด หาถิ่นฐานไม่ได้ อุบัติเหตุจากการเดินทาง |
| 丑 ฉลู, 戌 จอ, 未 มะแม | กลุ่มคลังธาตุพิฆาตกัน (ธาตุแฝงพิฆาต) | มี 2 ใน 3: ทำธุรกิจยาก; ครบ 3 = **ซำเฮ้ง** อุปสรรคมาก ลงทุนติดขัด (ยกเว้นวัยจรดี) ขัดแย้งภายใน ปัญหาเงินทอง/มรดก เก็บตัว ปัญหาระบบย่อย กล้ามเนื้อ เส้นเลือดตีบตัน เนื้องอก มะเร็ง |
| 子 ชวด, 卯 เถาะ | แม่ธาตุอิจฉากัน | ปัญหาความรัก ความสัมพันธ์ เซ็กซ์ เสน่ห์ ทำคุณคนไม่ขึ้น ถูกแกล้ง/อิจฉา ญาติพี่น้องไม่รัก |
| 辰 + 辰 | มังกร 2 ตัว (ธาตุแฝงเฮ้งกัน) | มั่นใจตัวเองสูงเกิน ยกตนข่มท่าน องุ่นเปรี้ยว ด่วนตัดสินใจ ไม่กลัวใคร ชอบเป็นปฏิปักษ์ |
| 午 + 午 | ม้า 2 ตัว (ธาตุไฟรุนแรง) | บุคลิกรุนแรง ก้าวร้าว ชอบโดดเด่น/มีชื่อเสียง หาเรื่องใส่ตัว ขี้ประชด ขัดแย้งเรื่องรัก อำนาจ หน้าที่ |
| 酉 + 酉 | ไก่ชน (แม่ธาตุทองรุนแรง) | ชอบหรูหรา บันเทิง เหล้า พนัน ชอบแย่งชิง จิกตีกัดไม่ปล่อย แย่งชิงความรัก |
| 亥 + 亥 | หมู 2 ตัว (แม่ธาตุน้ำมากไป) | หยุดความคิดไม่ได้ คิดเกินจำเป็น กังวลล่วงหน้า ยุ่งเรื่องคนอื่นจนเดือดร้อน คิดแทนผู้อื่น ตีความเอง |

## ไห่ — การให้ร้าย (เฉพาะราศีล่าง) — 6 คู่

ไม่ถูกกัน อึดอัดบอกไม่ถูก ถูกเกลียดชัง แทงข้างหลัง ชีวิตไม่มั่นคง แก้ปัญหารอบตัวยากขึ้น
**ถ้าชงร้ายด้วยเรื่องที่ชง จะแก้ยากมากหรือแก้ไม่ได้**

| คู่ไห่ | |
|---|---|
| 子 ชวด | 未 มะแม |
| 丑 ฉลู | 午 มะเมีย |
| 寅 ขาล | 巳 มะเส็ง |
| 卯 เถาะ | 辰 มะโรง |
| 申 วอก | 亥 กุน |
| 酉 ระกา | 戌 จอ |

## ผั่ว — การแตกแยก (เฉพาะราศีล่าง) — 6 คู่

ห่างเหิน ความสัมพันธ์จืดจาง ถูกกลั่นแกล้งจนเลิกลา เสียหาย ไม่มั่นคง ไม่คืบหน้า

| คู่ผั่ว | |
|---|---|
| 子 ชวด | 酉 ระกา |
| 丑 ฉลู | 辰 มะโรง |
| 寅 ขาล | 亥 กุน |
| 卯 เถาะ | 午 มะเมีย |
| 申 วอก | 巳 มะเส็ง |
| 未 มะแม | 戌 จอ |

หมายเหตุ: ขาล-กุน และ วอก-มะเส็ง เป็น **ภาคี (หลักฮะ) ด้วย** = เดี๋ยวรัก เดี๋ยวห่าง

## ซาฮะ — สามประสาน (Three Harmony)

ใช้เปรียบเทียบวัย: ดวงหลักใดก็ตาม + วัยจร + ปีจร/เดือนจร เจอครบ 3 ตัวตามแถว →
ติดป้าย [ซาฮะ] มีไอคอนไฟ

| | พาหะ | แม่ธาตุ | คลังธาตุ | แปรธาตุ |
|---|---|---|---|---|
| คู่กัน | 亥 กุน | 卯 เถาะ | 未 มะแม | **ไม้** |
| | 寅 ขาล | 午 มะเมีย | 戌 จอ | **ไฟ** |
| คู่กัน | 巳 มะเส็ง | 酉 ระกา | 丑 ฉลู | **ทอง** |
| | 申 วอก | 子 ชวด | 辰 มะโรง | **น้ำ** |

### ครึ่งซาฮะ (ครึ่งภาคี)

| ครึ่งแรก | ครึ่งหลัง | แปรธาตุ |
|---|---|---|
| 子 ชวด + 申 วอก | 子 ชวด + 辰 มะโรง | น้ำ |
| 酉 ระกา + 巳 มะเส็ง | 酉 ระกา + 丑 ฉลู | ทอง |
| 午 มะเมีย + 寅 ขาล | 午 มะเมีย + 戌 จอ | ไฟ |
| 卯 เถาะ + 亥 กุน | 卯 เถาะ + 未 มะแม | ไม้ |

## ซาหุย (ซาหวย) — สามฤดู (Seasonal Trio)

เงื่อนไขการเจอเหมือนซาฮะ (ครบ 3 → [ซาหุย] มีไอคอนไฟ)

| พาหะ | แม่ธาตุ | คลังธาตุ | แปรธาตุ |
|---|---|---|---|
| 亥 กุน | 子 ชวด | 丑 ฉลู | **น้ำ** |
| 寅 ขาล | 卯 เถาะ | 辰 มะโรง | **ไม้** |
| 巳 มะเส็ง | 午 มะเมีย | 未 มะแม | **ไฟ** |
| 申 วอก | 酉 ระกา | 戌 จอ | **ทอง** |

## หลักฮะ — ภาคีคู่ (Six Harmony / Secret Friend)

เงื่อนไขการเจอเหมือนซาฮะ แต่**ไม่ต้องมีไอคอนไฟ** → ติดป้าย [หลักฮะ]

| ภาคีคู่ | แปรธาตุ |
|---|---|
| 子 ชวด + 丑 ฉลู | ดิน |
| 寅 ขาล + 亥 กุน | ไม้ |
| 卯 เถาะ + 戌 จอ | ไฟ |
| 辰 มะโรง + 酉 ระกา | ทอง |
| 巳 มะเส็ง + 申 วอก | น้ำ |
| 午 มะเมีย + 未 มะแม | ไฟ |
